This website!

This portfolio website is a project in itself as it is hosted on the AWS cloud platform. The site showcases a number of different approach's when working with AWS.

  1. CICD is implemented using codepipeline attached to Github.
  2. The static site is hosted in a S3 bucket.
  3. The "contact me" form is implemented using API gateway linked to a Node.js lambda function.
  4. DNS routing and hosting is done through Route 53.

Honours Project

Object detection paired with temperature sensing for the blind.

I implemented a prototype on the raspberry PI hardware platform. The goal was to provide verbal assistance to people with visual impairments. To test this concept I narrowed the functionality down to only detecting if a household item like a coffee mug was in front of the user and If it was hot or not. The object detection was implemented using a TensorFlow lite neural network and the temperature sensing was built around a thermal camera module.
